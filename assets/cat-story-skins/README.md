# Storybook cat family assets

These are original AI-generated project assets created from the project's black-cat illustrations as visual style references. They are not copied from a third-party cat pack.

- `*-young-v2.png`: kitten form used during the first two growth stages.
- `*-adult-v2.png`: adult form used after the cat grows up.

Only the six production RGBA sprites are kept in the distributable project.
Intermediate atlases and superseded animation experiments were removed after
visual verification so the runtime contains one coherent cat family.
